Descripción:
Implementación básica de las funciones CRUD de la api REST de Django.

#Requisitos previos; tener instalado VSC, Python.
Instrucciones para instalar la app de forma local:
1.Descargar/Copiar la app subida en el GitHub https://github.com/SilverHopes/drfsimpletest
2.Extraer el archivo, abrir VSC, abrir el archivo descargado.
3.Abrir la terminal y escribir el comando python manage.py runserver
4.Copiar y/o abrir la dirección local entregada.
    #En caso de error, cerciorarse de tener instalado Django.
